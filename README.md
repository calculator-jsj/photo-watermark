# Photo Watermark

一个简单的命令行工具，用于给图片添加拍摄日期水印。  
程序会自动读取图片 **EXIF 信息** 中的拍摄日期（年月日），并在图片上绘制水印。  

---

## 功能特性
- 自动读取图片 EXIF 拍摄日期（年月日）
- 在图片上添加文字水印
- 支持用户自定义：
  - 字体大小
  - 字体颜色
  - 水印位置（左上 / 右上 / 居中 / 左下 / 右下）
- 处理后的图片会保存在 **原目录名 + `_watermark`** 子目录下
- 保留原始图片不修改
